Guess The Lyrics

How To Play

To start the game make sure you set the Timer!!!

  • To Guess Lyrics for a random Song: Just set the timer and hit Submit!
  • To Guess Lyrics for a random Song, but for a specific Artist: Just select the artist, and hit Submit!
  • To Guess Lyrics for a specific Song: Select both the artist and the song and hit Submit!
